# MCP Control Center 模块实施与验收清单

本清单是开发执行版文档。每个模块必须同时满足“能看到、能跳转、能操作、能反馈、能按权限限制”五项要求。

## 总览

- [x] 时间范围：7/15/30/90/180 天
- [x] Workspace 选择
- [x] MCP 总数、已上架、今日调用、成功率、响应时长、待处理错误指标
- [x] 指标跳转到 MCP 列表、调用分析、错误中心

## 发布前验收

1. 从总览指标完成一次跳转并验证目标筛选。
2. 从 Service 抽屉进入 Service 详情，再进入下属 MCP。
3. 从错误中心进入 MCP 报错分析，并查看上下文。
4. 完成上架五步向导并返回修改。
5. 在维护页完成 Ping 和异常详情查看。
6. 在市场完成接入详情、沙盒安装和审核提交。
7. 在知识库完成上传入口、详情、编辑和删除操作。
8. 切换只读角色，确认处理按钮不可用。
